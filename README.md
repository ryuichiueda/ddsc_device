# DDSC Device Driver for Linux

## Environment

- now only examined on Ubuntu 20.04 LTS

## How to use

	$ git clone https://github.com/ryuichiueda/ddsc_device.git
	$ cd ddsc_device
	$ make
	$ sudo insmod ddsc.ko
	$ sudo chmod 666 /dev/ddsc0
	$ cat /dev/ddsc0
	ドスコスコスコスコドドドドドドスコドドスコドドドドスコドド・・・

Enjoy.
